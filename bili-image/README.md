# bilibili image

该脚本用于B站图片的预处理，通过去除后缀的方式获取原始图像。

## 主要功能

- 右键打开图片时，去除图片修饰参数，获取原始图像。
- 可以通过油猴中的菜单选项开启或关闭该功能。

## 安装说明

需要先安装Tampermonkey（或其他支持 Userscript 的扩展）。具体步骤不做说明。
