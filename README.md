# bilibili avatar

之前B站可以通过右键头像的方式选择打开空间头像，在某段时间以后似乎就不可以这样操作。因此该脚本用于B站个人空间的头像扩展，让个人空间的头像可点击并操作（查看原图、下载、以图搜图）。

## 主要功能

- 在个人空间页将头像区域变为可点击元素。
- 点击后在新窗口打开一个简洁的展示页，支持：
  - 查看原始图片。
  - 一键下载头像图片。
  - 一键以图搜图（暂时使用 Google Lens）。

## 安装说明

需要先安装Tampermonkey（或其他支持 Userscript 的扩展）。具体步骤不做说明。
